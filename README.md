# Patchwork

## Overview
Among the scattered pieces of abandoned toys, a broken puppet finds itself thrown to the sewers. Collect the missing limbs to solve the puzzles and help him find his way out.

## Controls
- WASD = movement + minigame interaction
- Space = jump
- F = interactions with boxes, moving platform, and limbs
- Shift = charge boost
- R = reload level

## Gameplay
Novel Game Mechanic: Assemble your character using body parts scattered around the map. Each part allows the player to interact with the environment in new ways. 
- Traverse through the obstacles in the map
- Collect limbs and solve a mini puzzle to reattach
- Gain new abilities per limb

## Tech Stack and Tools
- Unity (2D)
- C#
- Krita (graphics)
